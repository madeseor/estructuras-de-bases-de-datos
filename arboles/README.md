# Árboles de Búsqueda

Esta carpeta contiene implementaciones de tres estructuras de datos de árboles utilizados para búsqueda eficiente de información.

## Contenido

| Código | Estructura | Descripción |
|--------|-----------|-------------|
| `ABB`  | Árbol Binario de Búsqueda | Árbol donde cada nodo tiene como máximo dos hijos, con el subárbol izquierdo menor y el derecho mayor |
| `B`    | Árbol B | Árbol balanceado y autoajustable diseñado para almacenar grandes volúmenes de datos |
| `B+`   | Árbol B+ | Variante del Árbol B donde todos los datos se almacenan en las hojas, optimizado para búsquedas por rango |
| `Comparador` | Comparador de Tiempos | Código que mide y compara los tiempos de ejecución de los tres árboles anteriores |

## Objetivo

Comparar el funcionamiento y rendimiento de estas tres estructuras al momento de realizar operaciones de **búsqueda**, **inserción** y **eliminación** de datos.

##  Comparador de Tiempos de Ejecución

Se incluye un **cuarto código** cuyo propósito es medir y comparar el rendimiento de los tres árboles. Este comparador:

- **Verifica previamente** que cada árbol esté funcionando correctamente antes de realizar cualquier medición
- **Mide los tiempos de ejecución** de las operaciones principales en cada estructura
- **Presenta los resultados** de forma comparativa para facilitar el análisis de rendimiento

> La verificación previa garantiza que los tiempos medidos reflejen el comportamiento real de cada estructura y no errores en la implementación.
