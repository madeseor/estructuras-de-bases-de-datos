# Estructura de Datos y Laboratorio
### Curso.

Este repositorio es un espacio de trabajo para los laboratorios y ejercicios prácticos desarrollados en el curso **Estructura de Datos y Laboratorio**. En él se documentan actividades relacionadas con el almacenamiento de datos, incluyendo diagramas, implementaciones y estructuras trabajadas a lo largo del curso.
